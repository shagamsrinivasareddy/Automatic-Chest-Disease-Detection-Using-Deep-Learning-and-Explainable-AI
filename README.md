# Automatic-Chest-Disease-Detection-Using-Deep-Learning-and-Explainable-AI
AI-powered chest X-ray analysis system for detecting thoracic diseases using ResNet50 and Grad-CAM visualization.

## Overview

This project presents a deep learning based system for automatic detection of thoracic diseases from chest X-ray images. The system uses a convolutional neural network architecture to identify multiple lung diseases and provides explainable results using Grad-CAM visualization.

The objective of this project is to assist medical professionals by providing AI-based diagnostic support for early detection of lung abnormalities.

## Key Features

* Deep Learning based chest disease detection
* Multi-label classification of thoracic diseases
* Explainable AI using Grad-CAM heatmaps
* Automated medical image analysis
* User-friendly interface for X-ray upload and prediction

---

## Technologies Used

* Python
* PyTorch
* ResNet50 (Transfer Learning)
* Grad-CAM
* Streamlit (User Interface)

---

## System Workflow

1. User uploads a chest X-ray image
2. Image is preprocessed and normalized
3. Deep learning model analyzes the image
4. System predicts possible thoracic diseases
5. Grad-CAM generates heatmaps highlighting affected regions

---

## Detected Diseases

The system detects the following thoracic diseases:

* Atelectasis
* Cardiomegaly
* Consolidation
* Edema
* Effusion
* Emphysema
* Fibrosis
* Hernia
* Infiltration
* Mass
* Nodule
* Pneumonia
* Pneumothorax
* Pleural Thickening

---

## Explainable AI

Grad-CAM visualization highlights the regions in the X-ray that influence the model's predictions, improving transparency and interpretability.
